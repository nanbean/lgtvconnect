LG TV Connect
=============

How to install
-------
```bash
yarn install
yarn build
```

How to develop
-------
```bash
yarn start
```
You need to change the config in `scr/service/firebase.js`

How to deploy
-------
```bash
yarn build
firebase deploy
```
You need firebase tools and firebase init.

[firebase:Deploy Your Site](https://firebase.google.com/docs/hosting/deploying)
